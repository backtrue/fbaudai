# 報受眾 (AUDAI) - AI 廣告受眾分析平台

## 📋 產品概述

**報受眾**是一個專業的 AI 驅動廣告受眾分析平台，專為電商賣家與廣告投放人員設計。透過先進的多模態深度學習技術，平台能夠分析產品圖片，自動識別目標受眾特徵，並基於真實 Facebook Graph API 數據生成精準的廣告受眾建議，幫助您大幅提升廣告投資報酬率（ROI）。

### 核心價值主張

- **🎯 精準定位**：利用 AI 技術深度分析產品特性，找出最有可能購買的目標客群
- **⚡ 快速高效**：只需上傳產品圖片，數秒內即可獲得專業的受眾分析報告
- **📊 真實數據**：所有受眾建議皆基於 Facebook Graph API 實際數據驗證
- **💰 提升 ROI**：降低廣告測試成本，縮短找到精準受眾的時間

## ⚙️ 核心功能

### 1. 智能產品分析

#### 功能描述
上傳產品圖片後，系統會運用 OpenAI GPT-4o Vision 多模態 AI 模型進行深度分析，自動識別：

- **產品名稱**：AI 自動辨識並命名產品
- **產品類別**：多層次產品分類（如：3C產品 > 手機配件 > 手機殼）
- **目標受眾**：潛在購買客群的人口統計特徵（年齡、性別、興趣等）
- **關鍵字標籤**：與產品相關的行銷關鍵字
- **信心分數**：AI 分析結果的可信度評分（0-100%）

#### 使用方式
1. 點擊「開始分析」進入分析頁面
2. 拖曳或點擊上傳產品圖片（支援 JPG、PNG 格式，最大 5MB）
3. 等待 AI 分析完成（通常 3-10 秒）
4. 查看分析結果並確認

#### 技術特色
- **多模態學習**：同時分析圖像特徵與文字內容
- **階層式分類**：採用深度學習分類系統，涵蓋超過 50 種產品類別
- **自適應優化**：持續學習並優化分析準確度

---

### 2. Facebook 廣告受眾建議

#### 功能描述
基於 AI 分析結果，系統會透過 Facebook Graph API 搜尋並驗證最適合的廣告受眾標籤，提供：

- **興趣標籤（Interests）**：用戶的興趣愛好與關注領域
- **行為標籤（Behaviors）**：用戶的購買行為與使用習慣
- **人口統計（Demographics）**：年齡、性別、地區等基本特徵
- **品牌標籤（Brands）**：相關品牌的粉絲群體

#### 建議資訊包含
- **受眾類型**：標籤所屬類別
- **受眾名稱**：Facebook 官方標籤名稱
- **受眾 ID**：可直接用於廣告投放的 Facebook Audience ID
- **預估觸及**：該受眾標籤的潛在觸及人數
- **驗證狀態**：是否已通過 Meta Graph API 驗證
- **使用建議**：如何在廣告投放中運用該標籤

#### 使用方式
1. 確認 AI 分析結果無誤
2. 可選填價格區間與銷售地區以精準化建議
3. 點擊「生成受眾建議」
4. 等待系統搜尋並驗證受眾標籤（約 5-15 秒）
5. 查看完整的受眾建議清單

#### 技術優勢
- **即時驗證**：所有建議皆透過 Facebook Graph API v19.0 即時驗證
- **動態估算**：提供受眾規模的即時估算
- **多重來源**：結合 AI 語意分析與 Facebook 官方數據

## 🛠 技術架構

### 前端技術
- **框架**：React 18 + TypeScript
- **UI 組件**：Radix UI + shadcn/ui
- **樣式系統**：Tailwind CSS
- **路由管理**：Wouter
- **狀態管理**：TanStack Query (React Query)
- **表單處理**：React Hook Form + Zod 驗證

### 後端技術
- **運行環境**：Node.js + Express.js
- **語言**：TypeScript (ESM modules)
- **資料庫**：PostgreSQL (Neon Serverless)
- **ORM**：Drizzle ORM + Drizzle Kit
- **認證系統**：JWT + Google OAuth 2.0
- **檔案處理**：Multer + Sharp

### AI 與外部服務
- **AI 模型**：OpenAI GPT-4o Vision API
- **廣告 API**：Meta Graph API v19.0
- **認證服務**：eccal.thinkwithblack.com Unified Auth
- **雲端託管**：Replit Deployment

### 安全性措施
- **JWT 令牌**：HTTP-Only Cookie，防止 XSS 攻擊
- **CORS 設定**：嚴格的跨域資源存取控制
- **檔案驗證**：圖片格式與大小限制（最大 5MB）
- **API 金鑰**：環境變數安全管理
- **資料加密**：HTTPS 傳輸加密

- **持續優化**：根據廣告數據調整受眾設定
- **參考觸及數**：平衡受眾規模與精準度

## 🆘 常見問題（FAQ）

### Q1：分析一次需要多久時間？
**A**：通常 3-10 秒即可完成 AI 分析，受眾建議生成約需 5-15 秒，取決於網路速度與 API 回應時間。

### Q2：支援哪些圖片格式？
**A**：支援 JPG、PNG 等常見圖片格式，單一檔案最大 5MB。

### Q3：受眾建議的準確度如何？
**A**：所有建議皆基於 OpenAI GPT-4o Vision 分析與 Facebook Graph API 官方數據，準確度通常在 85-95% 之間（以信心分數顯示）。

### Q4：可以匯出哪些格式的資料？
**A**：目前支援 JSON 格式匯出，包含完整的分析結果與受眾建議資訊。

### Q5：免費會員與付費會員的差異？
**A**：
- **Free**：每月 10 次分析
- **Pro**：每月 50 次分析 + 優先支援
- **Enterprise**：無限次分析 + API 存取 + 專屬客服

### Q6：如何升級會員方案？
**A**：前往「設定」頁面，在「訂閱方案」區塊點擊「升級為 PRO」即可。

### Q7：分析記錄會保存多久？
**A**：所有分析記錄永久保存，您可以隨時在「分析歷史」中查看。

### Q8：是否支援 API 串接？
**A**：Enterprise 會員可使用 API 進行系統整合，詳情請聯繫客服。
